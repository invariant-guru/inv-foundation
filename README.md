# foundation

DDD and Clean Architecture foundation library for NestJS applications. Provides base classes for entities, aggregate roots, value objects, repositories, error handling, logging, messaging, and testing utilities.

## Items

| Type        | Name                 | Description                                                        |
|-------------|----------------------|--------------------------------------------------------------------|
| Rule        | clean-architecture   | Clean Architecture layer dependency rules for the foundation library. |
| Context     | foundation-overview  | Overview of the foundation library structure and modules.          |
| Instruction | foundation-usage     | How to use the foundation library in NestJS projects.              |

## Install

```
invariant install @invariant--labs/foundation
```
